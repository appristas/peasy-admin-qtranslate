Peasy Admin qTranslate Field

== Description ==

This plugin adds qTranslate field to Peasy Admin. This plugin requires Peasy Admin v1.1.0.

= Resources =
* WordPress Plugin: https://wordpress.org/plugins/peasy-admin-qtranslate/
* Github Repository: https://github.com/appristas/peasy-admin-qtranslate
* Issues Tracker: https://github.com/appristas/peasy-admin-qtranslate/issues

== Installation ==

1. Install Peasy Admin: qTranslate either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin
3. Please visit the [plugin repository](https://github.com/appristas/peasy-admin-qtranslate) for usage.
